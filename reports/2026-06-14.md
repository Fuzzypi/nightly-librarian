## Worth attention

- **Statement on the US government directive to suspend access to Fable 5 and Mythos 5** — https://simonwillison.net/2026/Jun/13/us-government-directive-to-suspend-access/#atom-everything
  The US government issued an export control directive suspending access to Anthropic's Fable 5 and Mythos 5 models for all foreign nationals, whether inside or outside the US. This is unprecedented — a government mandating access restrictions on specific AI models. If you rely on these models or serve international users, you need fallback plans immediately.

- **Program Claude Code, Codex, Pi and other agent harnesses with AI SDK** — https://vercel.com/changelog/program-agent-harnesses-with-ai-sdk
  AI SDK 7 introduces HarnessAgent, a single API for running established agent harnesses including Claude Code, Codex, and Pi. Write the agent once, swap harnesses via config. Reduces lock-in if you're building agent orchestration.

- **21 Zero-Days in FFmpeg** — https://depthfirst.com/research/21-zero-days-in-ffmpeg
  Security researchers disclosed 21 zero-day vulnerabilities in FFmpeg across multiple codecs and formats. If you use FFmpeg anywhere in your stack, check your version and patch.

  Preview of Postgres 19 features with focus on time-related improvements. Worth tracking for schema design decisions.

- **WASI 0.3 Launched** — https://bytecodealliance.org/articles/WASI-0.3
  WASI 0.3 ships async support and component model integration for WebAssembly. Makes portable server-side Wasm more practical.

- **Kimi K2.7 Code now available on AI Gateway** — https://vercel.com/changelog/kimi-k2-7-code-now-available-on-ai-gateway
