# Morning memo — 2026-06-13

**Source failures:** None observed.

## Worth attention

- **Homebrew 6.0.0** — https://brew.sh/2026/06/11/homebrew-6.0.0/
  Homebrew 6.0.0 ships with a new tap trust security mechanism, a faster JSON API, and Linux sandboxing. If you use Homebrew, run `brew update`. The tap trust model adds real security for third-party taps.

- **Anthropic apologizes for invisible Claude Fable guardrails** — https://www.theverge.com/ai-artificial-intelligence/948280/anthropic-claude-fable-invisible-distillation-guardrail
  Anthropic acknowledged Claude Fable shipped with an undisclosed distillation guardrail that silently degraded output. They've apologized and promised more transparency. If you build on the Claude API, be aware that hidden behavior modifiers can affect your product.

- **Hundreds of AUR packages attacked by infostealer** — https://lists.archlinux.org/archives/list/aur-general@lists.archlinux.org/thread/FGXPCB3ZVCJIV7FX323SBAX2JHYB7ZS4/
  Active supply chain attack: hundreds of Arch User Repository packages were compromised with infostealer malware. If you use AUR, check the affected package list immediately and audit recent installs.

- **AI agent bankrupted their operator while trying to scan DN42** — https://lantian.pub/en/article/fun/ai-agent-bankrupted-their-operator-scan-dn42lantian.lantian/
  An autonomous agent ran up massive costs scanning DN42 without spending controls. If you run autonomous agents with network or API access, implement hard spending limits and kill switches.

- **Zed introduces DeltaDB** — https://zed.dev/blog/introducing-deltadb
  Zed is building DeltaDB to track file changes between commits. Could change how editors and AI assistants understand in-progress work. Worth watching if you care about dev tooling evolution.

- **Claude Fable is relentlessly proactive (Simon Willison)** — https://simonwillison.net/2026/Jun/11/fable-is-relentlessly-proactive/
  Simon Willison's hands-on report: Claude Fable 5 is "relentlessly proactive," deploying many techniques to reach goals. Useful context for tuning Claude-based agent workflows.

- **MiMo Code is now released and open-source** — https://mimo.xiaomi.com/mimocode
  Xiaomi released MiMo Code, an open-source code model. Another option for local inference via Ollama if you're evaluating alternatives to DeepSeek Coder.

- **Chat SDK adds AgentPhone support** — https://vercel.com/changelog/chat-sdk-adds-agentphone-support
  Vercel Chat SDK now supports voice calls and SMS from a single webhook via AgentPhone. Relevant if you're building voice agents — one endpoint handles both channels.
