## Worth attention

- **Mistral Vibe: unified agent with Work and Code modes, plus VS Code extension** — https://mistral.ai/news/vibe-agent
  Mistral launched Vibe as a unified long-horizon agent combining productivity (Work mode) and coding (Code mode), with a new VS Code extension shipping alongside. This is Mistral's direct entry into the coding agent space, competing with Cursor and Claude Code. The VS Code extension is the concrete actionable item — worth checking if evaluating Mistral's API for agentic coding workflows.

- **Custom MCPs in Mistral Studio** — https://mistral.ai/news/connectors
  Mistral Studio now supports built-in and custom MCP connectors with direct tool calling and human-in-the-loop approval controls. Directly relevant for MCP-based work. If you're already running MCP servers, Mistral is now a realistic option for connecting to them. Worth checking the connector spec for compatibility with existing MCP infrastructure.
